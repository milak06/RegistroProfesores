# Sistema de Registro de Profesores 📋

Aplicación web desarrollada como parte del Examen 1 del curso de Programación Orientada a Objetos II.

## Descripción
Sistema CRUD para gestionar el registro de profesores, permitiendo crear, listar, editar, ver detalles y eliminar registros.

## Tecnologías usadas
- ASP.NET MVC 5
- Entity Framework 6
- SQL Server
- C#
- Razor Views
- HTML5 / CSS3
- Bootstrap

## Funcionalidades
- ✅ Listar profesores
- ✅ Registrar nuevo profesor
- ✅ Editar datos de un profesor
- ✅ Ver detalles de un profesor
- ✅ Eliminar profesor con confirmación
